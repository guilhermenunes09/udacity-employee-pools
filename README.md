# Employee Polls Project

This Application allows different users to create pools for other users and vote.
They can choose between two options and all the answers and questions are recorded on Store.

At the Leadeboad page it's possible to visualize the users who have the most numbers of questions and answers.

## How to run the project

Make sure you have all requirements for running a React application on your local machine, then type these commands:

1.
```
  cd project-folder
```
2.
```
  npm install
```
3.
```
  npm start
```

## RUN TEST

Run test with the command below in other to verify the consistency of the components.

```
  npm run test
```
